Andrew's Directory App

Guide & Tutorial to this Express App:

This app supports the following functions:

    Register & Login Routes

  • Register as an Admin /register - (POST) with first_name (string), last_name (string), email_address (string), password (string)
  • Login as an Admin /login - (POST) with email_address (string), password (string)

    Company Routes

  • Create a Company - (POST) /company - with company_name (string)
  • Update Company Name (PATCH) /company/:company_id (PATCH) with company_name (string)
  • INTERNAL ONLY - List Available Companies - (GET) /company

    Add/Remove Admin To/From Company

  • Invite an Admin - (POST) /company/:company_id - with email_address (string)
  • Remove Admin from Company - (DELETE) /company/:company_id/remove/:admin_id

    Company Branches

  • Create a Company Branch - (POST) /company/:company_id/branch - with branch_name (string)
  • Update a Company Branch Name - (PATCH) /company/:company_id/branch/:branch_id - with branch_name (string)
  • Delete a Company Branch - (DELETE) /company/:company_id/branch/:branch_id
  • List Company Branches - (GET) /company/:company_id/branch
  • Show Specific Company Branch - (GET) /company/:company_id/branch/:branch_id
  • List Employees in Specific Branch - (GET) /company/:company_id/branch/:branch_id/directory

    Employee Calls

  • Detailed Information About a Specific User - (GET) /company/:company_id/branch/:branch_id/employee/:id
  • To Create a New User - (POST) /company/:company_id/branch/:branch_id/employee - with first_name (string), last_name (string), employee_email (string), job_title (string)
  • To Update an Existing User - (PUT) /company/:company_id/branch/:branch_id/employee/:id - with first_name (string), last_name (string), employee_email (string), job_title (string), is_active (boolean)
  • To Delete an Existing/Specific User - (DELETE) /company/:company_id/branch/:branch_id/employee/:id
